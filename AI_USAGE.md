# AI Usage Declaration

## Project: Gemstone Recommendation App

### AI Tools Used

| Tool | Purpose | Extent of Use |
|------|---------|---------------|
| **Gemini (Antigravity)** | Code generation, architecture design, debugging | Used throughout the project for generating boilerplate, components, styling, and backend logic |

### How AI Was Used

1. **Architecture & Planning** — AI assisted in designing the project structure, selecting the tech stack (React + Express + Python), and planning the recommendation algorithm.

2. **Backend Development** — AI helped generate the Express.js server setup, REST API routes, and the gemstone data model (gemstones.json).

3. **Python Recommendation Engine** — AI assisted in designing the weighted scoring algorithm for gemstone recommendations based on zodiac signs, life goals, budget, and color preferences.

4. **Frontend Development** — AI generated React components, page layouts, CSS styling (dark crystal theme with glassmorphism), and routing configuration.

5. **Data Curation** — AI helped compile the gemstone database with accurate properties, zodiac associations, planetary rulers, and metaphysical attributes.

6. **Documentation** — AI generated the README.md and this AI_USAGE.md file.

### Human Contributions

- **Project Requirements** — Defined the scope, features, and evaluation criteria
- **Review & Approval** — Reviewed AI-generated code, approved architecture decisions, and validated the final output
- **Deployment & Git** — Managed version control, commits, and GitHub deployment
- **Testing** — Manually tested the application flow and verified correctness

### AI-Generated vs Human-Written Code

| Component | AI Contribution |
|-----------|----------------|
| Project structure & config | ~90% AI-generated |
| Backend API (Express) | ~85% AI-generated |
| Python recommendation engine | ~80% AI-generated |
| Frontend UI (React + CSS) | ~85% AI-generated |

| Documentation | ~90% AI-generated |

### Disclaimer

While AI tools significantly accelerated development, all code was reviewed, tested, and validated by the developer before being committed. The architectural decisions and engineering choices reflect the developer's judgment and understanding of the requirements.
